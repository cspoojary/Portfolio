## 🚀 Getting Started

### Installation (3 steps)

1. **Upload Theme**
   ```
   /wp-content/themes/chaithanya-portfolio/
   ```

2. **Activate in WordPress**
   - Appearance → Themes → Activate

3. **Add Images**
   ```
   /wp-content/themes/chaithanya-portfolio/images/
   - github.png
   - linkedin-logo.png
   - email.png
   ```

See `QUICK_START.md` for fast 5-minute setup.

### Setup Content (10-15 minutes)

1. Create pages: Home, About, Projects, Experience, Certification
2. Add 5+ projects
3. Add 3+ certifications
4. Add 2+ experience entries
5. Create navigation menu
6. Test all links
